# Plagiarism Checker

A simple Python-based plagiarism checker that compares two pieces of text and reports a similarity score.  
It uses Python’s built-in `difflib.SequenceMatcher` to calculate a similarity ratio and prints a clear plagiarism detection report.

 Features

- Compares two text inputs
- Shows similarity score (0–100%)
- Classifies result as:
  - High similarity (possible plagiarism)
  - Moderate similarity
  - Low similarity
